# This is the frontend for multiplier app : https://github.com/amarg236/multiplier


## Steps tp run app:
- `git clone git@github.com:amarg236/multiplier-front.git`
- `cd multiplier-front`
- 'npm install`
- `npm start`

Now app will be available in : http://localhost:3000
